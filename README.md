# RulesIngestion

Tools and documentation for ingesting tabletop rules into the DungeonMind pipeline.

## Contents
- `Docs/` project notes, evaluations, and handoffs

## Entry Points
- `rules_ingestion_pipeline.py` - primary CLI pipeline (Marker -> enrich -> graph -> metrics)
- `ingestion_service.py` - FastAPI service wrapper for ingestion jobs
- `main.py` - thin CLI wrapper

## Getting started
Browse the `Docs/` folder for the current vertical-slice handoffs and pipeline notes.
